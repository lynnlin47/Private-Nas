# Private-Nas
Nas สำหรับคนงบประหยัด

วิธีใช้งาน

1. วิธีทำ autorun .bat
 1.ติดตั้ง xampp ถ้ายังไม่เคยติดตั้ง
 2.เข้าไปที่จุดที่ติดตั้ง ถ้า C ก็ C/xampp
 3.เลื่อนหาคำว่า apache_start.bat กดคลิ๊กขวา สร้างทางลัด
 4.ต่อไป กด windows + R และพิมพ์ shell:startup
 5.นำไฟล์ apache_start.bat มาไว้ในนี้ 
[เพียงเท่านี้ ตอนเปิดคอมใหม่ทุกครั้ง ไฟล์ xampp จะรันโดยอัตโนมัติโดยที่เราไม่ต้องมากดเปิดเองเลย แต่ ห้ามกดปิด ให้กดซ่อนหรือพับมันลงแค่นั้น ไม่งั้น server จะดับ]

2. วิธีทำให้ ip ในวงแลนใช้เว็บได้ (กรณ๊ใช้เน็ตตัวอื่นเชื่อมที่ไม่ใช่ Wifi)
 1.กด windows
 2.พิมพ์ Windows Defender Firewall with Advanced Security หรือ กด Windows Defender Firewall with Advanced Security ได้ในส่วนของ settings ของเว็บ
 3.กด Inbound Rule
 4.กด New Rule
 5.กด Port
 6.กด TCP และกด Specific local ports และใส่เลข 80, 443
 7.กด Allow the connection 
 8.กด Next และ ตังชื่อ เอาแบบง่ายๆ เช่น A_Host มันจะง่ายตอนค้นหา
 9.กลับไปหน้า Windows Defender Firewall with Advanced Security และ กด F5 
 [แค่นี้ กฏที่เราตั้งไว้ที่ชื่อ A-Host ก็จะอยู่ข้างบนสุด]

 โดยปกติแล้ว 127.0.0.1 ไม่สามารถให้เครื่องอื่นเข้าได้ ดังนั้นเราต้องใช้ ไอพีแลน
 โดยการไปกด IP-Lan ใน settings ของเว็บและหาคำว่า Wireless LAN adapter Wi-Fi และ หาคำว่า  IPv4 Address. . . . . . . . . . . : 192.168.**.***
 [และนำไอพีนั้น ไปให้คนในแลน เพื่อเข้าเว็บไซต์ได้]

 โปรดจำไว้ว่า Nas ตัวนี้ เหมาะกับการใช้ภายในบ้านเท่านั้น เนื่องจาก ไม่มีระบบป้อกันอะไรเลย ดังนั้นไฟล์อาจถูกขโมยได้หากเปิดแบบสาธารณะ
